Funktionen:

object GetRealContainer(object pOf)
Liefert den Behälter, in dem sich pOf befindet, wenn z.B. ein Clonk im Bombenschiff ist, und das Bombenschiff in der Werkstatt, dann wird die Werkstatt zurückgegeben.

int GetMiddlestVertex(object pObj)
Gibt die Vertexnummer des Vertexes zurück, der sich am nächsten zur Objektmitte befindet.

int GetSpeed(object pObj, int iPrecision)
Gibt die Geschwindigkeit in beide Richtungen zurück. Also nicht nur XDir oder YDir, so dass man mit der Richtung zusammen wieder XDir und YDir ausrechnen kann.
Benötigt, damit man mit dem Seil schwingen kann.

int GetMoveAngle(object pObj)
Liefert die Richtung in die sich pObj bewegt, als Winkel ausgehend von pObj

int GetActionTargetNum(object pObj, object pOfObj)
Gibt die Action Nummer an, die pObj bei pOfObj hat.
Bsp:
pOfObj->SetAction("Bla",pObj);
GetActionTargetNum(pObj, pOfObj); // = 0
oder:
pOfObj->SetAction("Bla",0,pObj);
GetActionTargetNum(pObj, pOfObj); // = 1
